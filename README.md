# ExRyze - Filament Base Projects

A reusable Laravel + Filament base system for building scalable internal products.

## Purpose

This project acts as a **foundation layer** for multiple systems:

-   EXCMS (Headless CMS)
-   Ticketing System
-   Club Management System

## Core Stack

-   Laravel 12
-   Filament 5 (Admin Panel)
-   Sanctum (API Authentication)
-   Filament Shield (RBAC)
-   Filament Breezy (Auth + 2FA)
-   Filament Curator (Media)

## Key Features

-   Modular architecture
-   Admin panel ready
-   API-ready (headless capable)
-   Role & permission system
-   Media management
-   Extensible plugin system
